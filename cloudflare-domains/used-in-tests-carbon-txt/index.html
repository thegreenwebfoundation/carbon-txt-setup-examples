Used in Tests

This site is a test site for hosting carbon.txt files, to simulate various scenarios.

  • carbon.txt - A simple, passing example of carbon.txt file.
  • multiple-exceptions.carbon.txt - A carbon.txt file that is intended to fail with multiple validation errors.
  • carbon-txt-with-csrd-and-renewables.txt - A simple example of carbon.txt file linking to a CSRD report containing renewables data.
  • carbon-txt-with-csrd-no-renewables-data.txt - A simple example of carbon.txt file linking to a CSRD to a CSRD report containing NO renewables data.
